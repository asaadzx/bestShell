# bestShell
 
this is simple C++ console-based shell that allows you to write some commands 

## Features 

- Write some command
- same of terminal

 ## Installation

 1. **clone the repository**

  ```bash
git clone https://github.com/asaadzx/bestShell
```
2. **compile the program:**

```bash
g++ -o myshell myshell.cpp
```

3. **Run the Program:**
   
```bash
./myshell
```
